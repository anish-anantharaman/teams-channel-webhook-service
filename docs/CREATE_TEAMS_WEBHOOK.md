# Creating an incoming webhook in Microsoft Teams Channel
<br />
<br />

- ### Select `...` in the channel for which you want the webhook, then select `Workflows` from the list of options.

![Channel Menu in Teams](docs/images/img1.png)

- ### This will open up a modal as shown below.

![View Modal](docs/images/img2.png)

- ### Go to the `Workflows` tab under `Your Workflows` section and select `View Templates`.

![Select Workflows section](docs/images/img3.png)

- ### This will show up all the workflows that Microsoft Teams offers. In that, select the workflow`Send webhook alerts to a channel`.

![Select channel notification workflow](docs/images/img4.png)

- ### Confirm the team and the channel under that team for which you are creating the workflow. Click on `Save`.

![Confirm the team and the channel](docs/images/img5.png)

- ### Copy the webhook link.
![Copy the webhook link](docs/images/img6.png)


## ⚠️ Keep this webhook URL secure, as anyone with it can post messages to your channel.
